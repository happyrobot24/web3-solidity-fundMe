# NPM 依赖安装规则

## 规则
所有包安装都应该使用 `--save-dev` 标志添加到 devDependencies 中，除非是生产环境必需的依赖。

## 安装命令格式

### 正确 ✅
```bash
npm install dotenv --save-dev
npm install lodash --save-dev
```

### 错误 ❌
```bash
npm install dotenv          # 会添加到 dependencies
npm install dotenv --save   # 同样添加到 dependencies
```

## 原因
- Hardhat 项目中，开发工具和脚本依赖都应该在 devDependencies 中
- 生产环境不需要这些开发工具
- 保持 package.json 结构清晰

## 当前项目依赖
